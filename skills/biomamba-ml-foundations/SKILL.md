# Biomamba ML Foundations

## Scope

Use this skill for the machine learning foundations notebook content: Python environment setup, visualization, sklearn workflows, model evaluation, regression, dimensionality reduction, clustering, classification, and text mining.

Do not use this skill for sequencing-specific workflows such as single-cell RNA-seq, bulk RNA-seq, SCENIC, hdWGCNA, or multiMiR.

## Workflow

1. Set up a reproducible Python or conda environment.
2. Load tabular data with pandas and inspect missing values, labels, and feature types.
3. Visualize data with matplotlib and seaborn.
4. Split data into train and test sets.
5. Build sklearn preprocessing pipelines with scaling and encoders as needed.
6. Choose supervised, unsupervised, or semi-supervised methods based on labels.
7. Evaluate with cross-validation, held-out metrics, and grid search.
8. For reporting, include model choice, feature processing, evaluation metric, and limitations.
