# General writing style and organization

## Storyline discipline

A statistical paper should read as one argument:

- Introduction defines the need and gap.
- Methods answer the gap.
- Results demonstrate evidence.
- Discussion closes the loop and states limitations.

## Structural expectations

- Keep headings informative and specific.
- Use one core idea per paragraph.
- Start paragraphs with topic sentences.
- Use transitions to reduce reader effort.

## Common annoyances to avoid

- Missing punctuation around displayed equations.
- Inconsistent notation for the same quantity.
- Manual figure/table numbering in prose instead of `\\ref`.
- Unordered or redundant keywords.
- Missing line numbers in review drafts.
- No reproducibility supplement plan.

## Figures and tables

- Prefer vector formats (`.pdf`, `.eps`) for line graphics.
- Reference every figure/table in text and interpret it.
- Keep captions self-contained.

## Notation hygiene

- Define symbols once.
- Use consistent math mode for symbols/signs.
- Avoid reusing symbols for different meanings.

## Quick self-check

- Can the paper be summarized as problem, gap, method, evidence, conclusion?
- Are all cross-references and citations clean?
- Is the manuscript readable without backtracking across sections?
