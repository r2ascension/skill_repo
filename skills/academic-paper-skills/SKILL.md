# academic-paper-skills

Use this collection router when:
- the user is planning a paper, designing an outline, or asking whether an academic idea is original
- the task is about turning an outline into manuscript prose or running a structured composition workflow
- the user names this collection or asks for an academic-paper planning or composition skill

Routing guidance:
- Route to `academic-paper-skills/strategist` for topic framing, gap finding, originality checks, and outline design.
- Route to `academic-paper-skills/composer` for writing the actual manuscript from a plan or outline.
- If the manuscript is statistics-heavy or ML/AI-specific, prefer the corresponding specialized writing skills instead.

Merge and precedence notes:
- Treat this as a planning/composition family, not the default choice for every paper-writing request.

Examples in this collection:
- academic-paper-skills/strategist
- academic-paper-skills/composer
- help me plan a paper outline before drafting
