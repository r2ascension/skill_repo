# paper-dissemination-router

Use this router when:
- the user wants a paper turned into slides, a poster, a website, a video abstract, or another dissemination asset
- the task is about presenting or promoting an existing paper rather than drafting the manuscript

Routing guidance:
- Use `academic-slides-router` for talks and presentations.
- Use `paper-2-web` for websites, videos, and posters built from a paper source.
- Use `nature-paper2ppt` when the user explicitly wants a Nature-style Chinese paper presentation workflow.
